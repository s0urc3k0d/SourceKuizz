# Documentation SourceKuizz

Cette section contient les documents de référence pour l'architecture, la sécurité, la gamification, la roadmap produit et les aspects temps réel de la plateforme SourceKuizz.

## Structure

- `architecture/` : Architecture logique, physique, modèles de données, flux.
- `architecture/real-time/` : Protocoles temps réel et performance.
- `security/` : Stratégies de sécurité et anti-triche.
- `gamification/` : Mécaniques de gamification et fonctionnalités ludiques.
- `product/` : Spécifications fonctionnelles (hôte / joueur), UX.
- `roadmap/` : Roadmap incrémentale.

## Priorités Actuelles (MVP)

1. Auth (pseudo/password + Twitch)
2. CRUD Quiz + Questions (MCQ)
3. Lancement / gestion d'une session
4. WebSocket basique (join, question, answer, reveal, scoreboard)
5. Scoring simple (correct + vitesse)
6. SQLite (migration future PostgreSQL)

> Pour approfondir chaque sujet, consulter les fichiers dédiés.
